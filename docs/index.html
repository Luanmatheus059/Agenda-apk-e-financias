Entradas, saídas, captura de bancos
⬇️ Baixar APK · 2.9 MB
  1. Mesmo processo do APK acima
  2. Depois de instalar: aba 🔔 Bancos → Abrir Configurações → ative Minhas Finanças na lista de "Acesso a notificações"
  3. Pronto. PIX, compras e transferências aparecem automaticamente.
📌 iOS não permite instalar app fora da App Store. Mas dá pra instalar como PWA (Progressive Web App): funciona igual a um app nativo, com ícone na tela inicial, abre offline, recebe notificações no iOS 16.4+.

Abra no Safari → Compartilhar → Adicionar à Tela
🚀 Abrir Agenda no Safari
  1. Toca no link acima (precisa ser no Safari, não funciona em outro browser)
  2. Toca no ícone Compartilhar (quadradinho com seta pra cima, embaixo)
  3. Rola e toca em Adicionar à Tela de Início
  4. Confirma. Vai aparecer o ícone azul "A" na tela inicial.
  5. Abra pelo ícone, não pelo Safari (assim funciona em tela cheia, sem barra do navegador)

🚀 Abrir Finanças no Safari
